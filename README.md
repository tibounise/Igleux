## What's inside ?

 * Digital
     * Texas Instruments DAC8830
 * Power ICs
     * Advanced Monolithic Systems AMS1117 (SOT223 package only)
 * RF
     * HM-10 Bluetooth module
 * Heatsinks
     * Intel LGA775 heatsink
